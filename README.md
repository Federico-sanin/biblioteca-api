# 📚 Biblioteca API

API REST desarrollada en ASP.NET Core para la gestión de una biblioteca.

## 🚀 Tecnologías
- .NET / ASP.NET Core
- Entity Framework Core
- SQL Server

## ⚙️ Configuración

1. Crear base de datos:
   - Ejecutar script SQL (si aplica)

2. Configurar conexión en `appsettings.json`:
   ```json
   "ConnectionStrings": {
     "DefaultConnection": "Server=.;Database=BibliotecaDB;Trusted_Connection=True;"
   }
